🔍 Что делает программа
VankaBravo Chat — это десктопное приложение для локального запуска больших языковых моделей в формате GGUF (например, Qwen2.5-Coder-7B). Программа работает полностью офлайн, без отправки данных в интернет.

Основные возможности:
Загрузка GGUF моделей:

Поддержка моделей в формате GGUF (например, Qwen2.5-Coder)

Работа через библиотеку llama.cpp

Поддержка GPU ускорения (Vulkan)

Управление диалогами:

Множество отдельных чатов

Автоматическое сохранение истории

Удаление ненужных диалогов

Настройка параметров:

Регулировка контекста (4k - 131k токенов)

Выбор языка ответов

Переключение CPU/GPU режима

Юридическая часть:

Встроенные лицензии MIT, Apache 2.0

Пользовательское соглашение

Система подтверждения принятия лицензий

Мониторинг ресурсов:

Отображение использования RAM

Контроль потребления памяти

Технические детали:
Использует llama-cpp-python для работы с моделями

Хранит чаты в JSON файлах (папка chats/)

Сохраняет подтверждения лицензий (папка licenses/)

Многопоточная генерация ответов (интерфейс не зависает)

📦 Требования
Windows (exe-файл собран под Windows)

Оперативная память: от 8GB (рекомендуется 16GB+)

Видеокарта: с поддержкой Vulkan (опционально, для ускорения)

Место на диске: ~7GB для модели + место для чатов

⚠️ Примечание
Программа не включает в себя файлы моделей. Вам нужно скачать GGUF модель отдельно (например, с Hugging Face) и указать путь к ней при загрузке.
